Prototipo web para el control y mantenimiento de vehiculos en un taller de mecanica automotriz

¿QUE HACEMOS?

La funcionalidad, o el uso que se le dará a nuestra pagina web, será el monitoreo en tiempo real de vehículos que hallan ingresado a un taller de mecánica automotriz.

Esta página web beneficiará de igual manera a trabajadores y solicitantes de servicios, ¿por qué?, porque el proceso de envío y recepción de información será más fácil, existirá la reserva de algún servicio, entre otros.

Se creará un prototipo web para el control y el mantenimiento de vehículos en un taller de mecánica automotriz en el que se verán beneficiados tres roles, administrativo, trabajador y usuario, cada uno de estos roles podrá realizar acciones diferentes que en conjunto conforman el proceso de desarrollo de diferentes acciones.
